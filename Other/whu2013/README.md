B.Beautiful Cities
题意：n个城市修地铁，每个城市的出度不大于1，求方案数
思路：枚举环的长度下，其余的是prufer序列数
分成 环的长度为n-i，其余点为i
环中的圆排列为 (n-i-1)!
prufer序列中 n^(i+1-2) 多加一个点是因为有一个点挂在环上
ans= C(n,i) * (n-i-1)! * n^(i-1) * (n-i) 
   = n!/i! * n^(i-1)
   = (n-1)! * Inverse(i!) * n^i
